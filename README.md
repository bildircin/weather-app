# Hava Durumu Web Uygulaması

Reactjs ile geliştirilen bu uygulamada, react hookları, redux toolkit i ve tailwind css kullanıldı. Anasayfada tarayıcı lokasyonundan alınan enlem ve boylam bigileri google apileri kullanılarak şehir, ülke adı gibi bilgiler elde edildi. Daha sonra hava durumu için https://collectapi.com/ adresinden bilgiler elde edildi.
